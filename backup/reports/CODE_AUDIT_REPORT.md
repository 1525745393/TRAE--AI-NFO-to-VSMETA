## 6. 建议修复列表

### 优先级 1 - 重要修复

1. **MD5 哈希使用**（高风险）
   ```python
   # 原代码
   hashlib.md5(f.read()).hexdigest()
   
   # 修复为
   hashlib.md5(f.read(), usedforsecurity=False).hexdigest()
   # 或使用 SHA-256
   hashlib.sha256(f.read()).hexdigest()
   ```

2. **裸异常捕获**（中风险）
   ```python
   # 原代码
   except Exception:
       pass
   
   # 修复为
   except Exception as e:
       logger.warning(f"处理失败: {e}")
   ```
