## 版本指南

- **主版本号**: 不兼容的 API 变更
- **次版本号**: 向后兼容的功能新增
- **修订号**: 向后兼容的问题修复

### 版本状态

- **开发中 (Development)**: 可能会有 breaking changes
- **测试版 (Beta)**: 功能完整，稳定版本前
- **稳定版 (Stable)**: 推荐用于生产环境
